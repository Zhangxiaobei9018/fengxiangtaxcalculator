# 连续劳务个税计算器

基于2025年个税新规的连续劳务报酬个人所得税计算器。

## 功能特点

- 📊 支持连续月份个税累计计算
- 💰 支持默认收入和按月单独输入
- 📱 响应式设计，支持移动设备
- 🎯 计算结果实时高亮显示
- 📖 详细的计税依据说明

## 使用方法

1. 设置默认月收入或使用"月金额不同点我计算"
2. 选择需要计算的月份
3. 点击"开始计算"查看结果
